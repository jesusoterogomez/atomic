A Modern UI Collection for Web Developers

  • What is Atomic?

    Atomic is a new collection of UI elements that will give your web projects a fresh, minimal look, while offering you a set of beautifully designed, user experience-oriented components, so you can focus on what matters the most: developing quick and effectively

  • Getting Started

    Kickstarting your project with Atomic couldn't be simpler, either download the library or import it with bower into your project using

    bower install atomic --save

    Reference atomic.css and atomic.js in your code and you're all set up.
